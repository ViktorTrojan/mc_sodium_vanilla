# Ignore Rules

Files and directories that AI agents should completely ignore and never read or modify:

```ignore
# Cache & output directories
node_modules/
.turbo/
dist/
build/

# Secrets
.env
.env.local

# Build artifacts & lock files
tsconfig.tsbuildinfo
bun.lock

# IDE configuration
.vscode/
.idea/

# AI related
.ai/generic/
```
